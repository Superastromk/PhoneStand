# PhoneStand Basic Description
A phone stand with space for wirelessly charging a galaxy watch and pair of galaxy buds. 3D printed.
# Why I Made This
I have a habit of losing my devices VERY easily, and I also have to keep moving from place to place to find the appropriate charger for each device. I made this so that all the chargers I would need are in one convenient spot, and I can also use it as a phone stand on my desk.
